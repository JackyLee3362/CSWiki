### 树的应用——并查集

- Union(S, Root1, Root2)
- Find(S, x)
- Initial(S)
- 并查集的结构定义

```c
#define SIZE 100
int UFSets[SIZE];
```

- 并查集的初始化操作

```c
void initUFSets(int S[]){
for(int i = 0; i < SIZE; i++)
S[i] = -1;
}
```

- 并查集的查找操作

```c
int Find(int S[], int x){
if(S[x] >= 0) x = S[x];
return x;
}
```

- 并查集的合并操作

```c
void Union(int S[], int Root1, int Root2){
S[Root2] = Root1;
}
```
