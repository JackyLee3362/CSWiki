## 8.2 插入排序

### 直接插入排序 Insert Sort

代码实现

```cpp
void InsertSort(int A[], int n) {
int i, j;
for (i = 2; i <= n; i++)
    if (A[i] < A[i - 1]) {
        A[0] = A[i];
        for (j = i - 1; A[0] < A[j]; j--)
            A[j + 1] = A[j];
        A[j + 1] = A[0];
    }
}
```
