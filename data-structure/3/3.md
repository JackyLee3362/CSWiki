- 栈在表达式求值中的应用
  - 前缀表达式
  - 中缀表达式
  - 后缀表达式
- 栈在递归中的应用
  - 斐波那契数列

```cpp
// 斐波那契数列的实现
int Fib(int n){
    if(n==0) return 0;        // 边界条件
    if(n==1) return 1;        // 边界条件
    return Fib(n-1) + Fib(n-2); // 递归表达式
}
```

- 递归模型必须满足下列两个条件
  - 递归表达式（递归体）
  - 边界条件（递归出口）
