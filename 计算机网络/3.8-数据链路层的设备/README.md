## 3.8数据链路层的设备
- 网桥
    - 透明网桥
        - 选择的不是最佳路由
    - 源路由网桥
        - 选择的是最佳路由
- 局域网 交换机
    - 虚拟局域网 VLAN
    - VLAN不仅不可以隔离冲突域，还可以隔离广播域
    - 两种交换模式
        - 直通式交换机
        - 存储转发式交换机
