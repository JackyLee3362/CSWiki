# 5.2 UDP(User Data Frame Protocol, 用户数据报协议)

## 概述

- UDP 仅在 IP 的数据报服务至上增加了 复用分用 差错检测（无拥塞控制）
- 无需建立连接
- UDP 首部格式
  - 首部开销为：8B
  - 源端口
  - 目的端口
  - 长度
  - 校验和

## UDP 校验

- 增加 12B 伪首部
- 校验首部和数据部分
- 注意：IP 数据报的校验和只校验 IP 数据报的首部，但 UDP 的校验和则检查首部和数据部分
- [二进制反码运算](https://www.cnblogs.com/jcchan/p/10400504.html)
- 接收端的多路分用
- 发送端的多路复用
