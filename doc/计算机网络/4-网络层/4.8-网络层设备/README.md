## 4.8 网络层设备
### 路由器的组成和功能 Router
- 组成
    - 交换结构
    - 一组输入端口
    - 一组输出端口
- 功能
    - 分组转发
    - 路由计算
- 路由表 路由转发
- 路由表是根据路由选择算法得出的
- 与网桥的区别
    - 网桥与高层协议无关，而路由器是面向协议的
- 路由表：| 目的网络IP地址 | 子网掩码 | 下一跳IP地址 | 接口 |
